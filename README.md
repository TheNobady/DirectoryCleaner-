# Directory Cleaner CLI 

A lightweight, Java-based command-line tool designed to organize messy directories by filtering and moving specific files into custom subdirectories.

##  How It Works
The program follows a simple 3-step logic to clean your workspace:
1. **Target**: You provide the absolute path of the messy directory.
2. **Setup**: You name a new subdirectory to hold the filtered files.
3. **Filter**: You provide a keyword or extension, and the tool moves all matches.



##  Prerequisites
- **Java Runtime Environment (JRE)**: Version 17 or higher recommended.
- **IDE**: Eclipse (configured with a valid JDK) or any terminal.
